ℹ About FileLite

FileLite is a secure cloud file sharing platform built with end-to-end encryption. Every file you upload is encrypted directly in your browser before it reaches our servers — meaning nobody, not even we, can read your files.

Files are encrypted using AES-256-GCM and keys are protected with RSA-2048 (OAEP). Your private key never leaves your device.

❓ How To Use

  • Register: Create an account — an encryption key pair is generated automatically
  • Upload: Select files (up to 400MB total) — they are encrypted before uploading
  • Share: Click Share on any file, type the exact username of the recipient, click Add then Share
  • Bulk Share: Click "Bulk Share Files", select files, add up to 10 users, click Share All
  • Download: Click Download — file is decrypted in your browser and saved to your device
  • Delete: Click Delete on a file — it is permanently removed for you and all recipients
  • Bulk Delete: Click "Bulk Delete Files", select files, confirm — they disappear one by one
  • Duplicates: If you select files already uploaded (same name + size), you'll be warned before uploading

💬 Frequently Asked Questions

Can anyone read my files?

No. Files are encrypted in your browser before upload. Even the server and database admin cannot read them.

What happens if I clear my browser data?

Your private key is stored in localStorage. If you clear browser data, your key is lost and you won't be able to decrypt your files. You would need to re-register.

Can I use FileLite on multiple devices?

Currently each device needs its own registration as the private key is stored locally. Multi-device support is not available yet.

What happens when the owner deletes a shared file?

The file is immediately removed from the recipient's "Shared With Me" section. If they already downloaded it, their local copy remains on their device.

How does duplicate detection work?

Before uploading, FileLite checks if any selected file has the same filename AND same file size as an existing uploaded file. Both conditions must match to be flagged as a duplicate.

Can I share with someone who is not registered?

No. The recipient must have a registered FileLite account. You must type their exact username.

Is there a file size limit?

Yes — 50MB per individual file and 400MB total per upload session or bulk share.

How many users can I share a file with?

Maximum 10 users per share operation (single or bulk).
