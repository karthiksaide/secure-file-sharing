How do I get started with FileLite? ▶
Register an account — a secure RSA-2048 key pair is generated automatically in your browser. Your private key is encrypted with your password and stored safely. Then login, upload files and start sharing.
How do I share a file? ▶
Click the Share button on any file. Type the exact username of the recipient and click Add. You can add up to 10 users. Then click Share. The file key is re-encrypted for each recipient automatically.
How do I upload multiple files? ▶
Click Choose Files and select multiple files at once. The total size must be under 400MB. Each file is encrypted individually before uploading. Progress is shown file by file.
Can anyone read my files? ▶
No. Files are encrypted in your browser using AES-256-GCM before they reach the server. Even we cannot read them. Only you and the users you share with can decrypt files.
How is my private key protected? ▶
Your private key is encrypted with your password using AES-256-GCM and PBKDF2 (100,000 iterations) before being stored on the server. Nobody can read it without your password — not even us.
What if I clear my browser data? ▶
No problem at all! Your encrypted private key is stored on our server. Just login again on any device — your password decrypts the key automatically and everything works perfectly.
Can I use FileLite on multiple devices? ▶
Yes! FileLite supports login from any device — phone, laptop, tablet, any browser. Just login with your username and password. Your encrypted private key is fetched from the server and decrypted using your password automatically. No setup needed.
Does it work across different browsers? ▶
Yes. Chrome, Firefox, Safari, Edge — all work. Login from any browser, your key is always available from the server.
What if I forget my password? ▶
Currently there is no password reset feature. Since your private key is encrypted with your password, forgetting your password means you cannot decrypt your files. Keep your password safe.
How many users can I share a file with? ▶
Maximum 10 users per share operation, whether single file share or bulk share.
Can I share with someone not registered? ▶
No. The recipient must have a registered FileLite account. You must type their exact username. If the username does not exist, you will get an error.
What is bulk share? ▶
Bulk share lets you select multiple uploaded files and share them all to multiple users at once. Max 80 files, max 10 users, max 400MB total.
What are recent recipients? ▶
When you open the share modal, users you have previously shared files with appear as quick-select chips. Click them to add without typing again.
What is the maximum file size? ▶
50MB per individual file. 400MB total per upload session or bulk share operation.
How does duplicate detection work? ▶
Before uploading, FileLite checks if any selected file has the same filename AND same file size as an already uploaded file. Both conditions must match to be flagged as a duplicate. You can remove duplicates with one click before uploading.
What is the bulk share file limit? ▶
Maximum 80 files per bulk share operation. If you try to select an 81st file, it is automatically blocked. Total size of selected files must also be under 400MB.
Can I upload any file type? ▶
Yes, any file type is supported — PDFs, images, documents, videos, zip files, etc. All are encrypted the same way regardless of type.
What happens when I delete a shared file? ▶
The file is permanently deleted from storage and immediately disappears from all recipients' "Shared With Me" sections. This cannot be undone.
Can I recover a deleted file? ▶
No. Deleted files are permanently removed and cannot be recovered. If a recipient downloaded the file before deletion, their local copy remains on their device but the file is gone from the app.
